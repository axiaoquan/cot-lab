## 0. Why this project

Most CoT explainers either stop at "use this prompt" or jump straight to "train a 70B model on 1024 H100s". This project lives in the middle: implement each method on a small open model, run it on the same dataset, and compare.

You don't need a cluster. You need a Mac.
